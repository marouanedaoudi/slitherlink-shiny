# Package index

## Grid

Create and manipulate puzzle grids.

- [`init_grid()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/init_grid.md)
  : Initialize a Slitherlink grid
- [`toggle_segment()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/toggle_segment.md)
  : Toggle a segment in a Slitherlink grid

## Puzzles

Built-in puzzle library.

- [`list_puzzles()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/list_puzzles.md)
  : List available predefined puzzles
- [`get_puzzle()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/get_puzzle.md)
  : Load a predefined puzzle as a slitherlink_grid

## Validation

Check constraints and loop topology.

- [`check_clues()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/check_clues.md)
  : Check if all clues are satisfied
- [`check_loop()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/check_loop.md)
  : Check if drawn segments form a single closed loop
- [`is_solved()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/is_solved.md)
  : Check if a Slitherlink grid is fully solved

## Solver

Automatic puzzle solver.

- [`solve_grid()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/solve_grid.md)
  : Solve a Slitherlink puzzle

## Random puzzle

Generate random solvable puzzles.

- [`random_puzzle()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/random_puzzle.md)
  : Generate a random solvable Slitherlink puzzle

## Shiny app

Launch the interactive application.

- [`run_app()`](https://marouanedaoudi.github.io/slitherlink-shiny/reference/run_app.md)
  : Launch the Slitherlink Shiny application
